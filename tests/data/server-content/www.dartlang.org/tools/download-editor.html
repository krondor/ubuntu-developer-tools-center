Getting Dart Is Easy!

Download Dart

The Dart Editor bundle is the easiest way to get started with Dart. When you download the Dart Editor bundle, you get everything you need to create, edit, test, and build both web-based and server-side applications.

Current stable version of Dart: [calculating]

Important: Dart Editor for Linux is only for Ubuntu. If you use Debian or any other non-Ubuntu distribution, you should download just what you need and choose another editor.

The Dart Editor bundle includes:

  • Dart Editor: A powerful tool for editing, debugging, and running Dart applications.
  • Dartium: A special version of Chromium that includes the Dart VM. You can use Dartium to test and debug your Dart web applications.
  • Dart SDK: The Dart VM, libraries, and command-line tools such as dart, dart2js, dartanalyzer, pub, and dartdocgen.

Tip: To enable automatic updates of Dart Editor and Dart, go to the Dart Editor Preferences, choose Update, and select Download updates automatically. For more information, see the documentation for Dart Editor.

Download Dartium

If you’re writing a web app, we recommend downloading Dartium. Dartium is a special build of Chromium that includes a Dart VM so that you can test and debug your web app without first compiling it to JavaScript.

The Dartium binary expires after 12 weeks. When that happens, you need to download a new copy to continue using Dartium.

Or download just what you need

Step 1: Install Dart

We recommend using Homebrew to manage your Dart installation. To install Dart, run:

$ brew tap dart-lang/dart
$ brew install dart dartium

The dart package contains the Dart SDK, which includes the Dart VM, libraries, and command-line Dart tools such as dart, dart2js, dartanalyzer, pub, and dartdocgen.

The dartium package contains a special build of Chromium (nicknamed Dartium) that includes a Dart VM. Use it to test and debug Dart web apps without first compiling them to JavaScript.

Writing server-side code? If you don’t write Dart web apps, you don’t need to install dartium.

To update the software later, run:

$ brew update
$ brew upgrade dart dartium

Option: Use Homebrew with a custom location

By default, Homebrew downloads to /usr/local. This requires using sudo to handle issues with file permissions. If you prefer, you can download to another location where you have write permissions, such as your home directory. The following example creates a copy of Homebrew and Dart under ~/homebrew.

$ cd ~
$ git clone https://github.com/Homebrew/homebrew.git

Add ~/homebrew/bin to your PATH.

Then follow the brew instructions at the beginning of Step 1. Dart is installed under ~/homebrew.

Option: Download Dart manually

An alternative to using Homebrew or Dart Editor is downloading Dart manually. You’ll need to perform the following steps periodically to update your software.

  1. Download the Dart SDK. If you need a bigger heap or access to much more memory, download the 64-bit Dart SDK. Unzip the downloaded file to get the SDK.

  2. Add <sdk>/bin to your PATH.

  3. If you’re developing a web app, download Dartium. Unzip it to get special version of Chromium with the Dart VM. This binary expires after 12 weeks. When that happens, you need to download a new copy to continue using Dartium.

Step 1: Install Dart

Choose either:

  • Server-side development for Debian/Ubuntu
  • Other Linux development

If you want to build Dart from scratch, see Compiling from source.

Installing for server-side development on Debian/Ubuntu

Get a 64-bit version of the Dart SDK that’s easy to update:

# Enable HTTPS for apt.
$ sudo apt-get update
$ sudo apt-get install apt-transport-https
# Get the Google Linux package signing key.
$ sudo sh -c 'curl https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -'
# Set up the location of the stable repository.
$ sudo sh -c 'curl https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list'
$ sudo apt-get update
$ sudo apt-get install dart

For more information about other options for Debian, visit the Dart Debian downloads page.

Compiling from source

You can build the SDK yourself. If your system uses an older version of Ubuntu, you might need to update to GCC 4.6 or later. See the Tools FAQ for more information.
